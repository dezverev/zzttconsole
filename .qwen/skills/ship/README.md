# Ship - Automated Git Workflow

This skill automates the complete Git workflow for features and fixes.

## What It Does

1. Checks if you're on `main`/`master`
2. Creates a new feature branch
3. Stages and commits your changes
4. Pushes the branch to remote
5. Creates a pull request
6. Merges the PR

## Prerequisites

- Git CLI
- GitHub CLI (`gh`) for automated PR creation

## Usage

Ask Qwen Code to "ship" your changes and it will handle the entire workflow automatically.
